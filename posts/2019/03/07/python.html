# 定时任务

python 标准库 sched 可实现任务定时运行：

import sched
import time
from datetime import datetime
def perform_task():
    scheduler.enter(60*60*24, 0, perform_task)
    print('hello world')
if __name__ == '__main__':
    scheduler = sched.scheduler(time.time, time.sleep)
    now = datetime.now()
    sched_time = datetime(now.year, now.month, now.day, 4, 0, 0)
    if sched_time < now:
        sched_time = sched_time.replace(day=now.day+1)
    scheduler.enterabs(sched_time.timestamp(), 0, perform_task) 
    print('crontab run')
    scheduler.run()

上面的程序实现了每天凌晨4点执行perform_task任务。

第10行构造了一个sched.scheduler类，该类接收两个参数:(timefunc, delayfunc) , timefunc 需要返回一个数字，代表当前时间；delayfunc 接受一个参数，用于暂停运行的任务。一般使用 time.time 和 time.delay 来完成构造。

scheduler 类提供两种主要的任务：

  • 绝对时间任务：scheduler.enterabs(time, priority, action, argument=(), kwargs={}) 其中 time接受一个日期的 timestamp 时间戳，priority为当前任务优先级， action为执行函数， argument 为action所需要的参数
  • 相对时间任务：scheduler.enter(delay, priority, action, argument=(), kwargs={}) 其中delay接受一个整数，表示 delay 秒后开始运行，其余参数与上面相同。

注意

  1. action可以套用一个多线程wrapper，防止action运行时间过长导致其后定时任务不能按期执行
  2. 一般不在action中写死循环，以防止引发线程阻塞或线程灾难。如果要不断重复运行任务，可以在action中再次添加 scheduler.enter 来实现定时重复。（如上例）
